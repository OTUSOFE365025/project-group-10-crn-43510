# AI Digital Assistant Platform (AIDAP) – Phase II

## Project Overview

The AI Digital Assistant Platform (AIDAP) is designed to help students, faculty, and administrators access university data using a conversational interface. Key features include course information, deadlines, announcements, analytics, and integration with university systems (LMS, calendar, email).

Stakeholders: Students, Lecturers, Administrators, Maintainers
