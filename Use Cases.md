# Use Cases 


| **Use Case ID** | **Use Case Name** | **Description** |
|-----------------|-------------------|-----------------|
| **UC-1** | **Ask Academic Question** | A user interacts with the AI assistant to ask academic or administrative questions (e.g., "When is my next exam?"). The system processes the query, retrieves data from connected systems, and provides a natural-language response. |
| **UC-2** | **Personalized Dashboard** | The System should give students acccess to a personalized dashbaord which would summarizer upcoming events (e.g., Assingemnts, exams), and performancxe indicators such as current overall grade or GPA. |
| **UC-3** | **Change of Pereference for Notifications & Language** | Students should be given the option to chnage thier perfeerences for the notifications they receicve and the primary interaciton language. |
| **UC-4** | **Calendar Export** | Students should be provided a way in order to export their academic calendar events to personal calendars (e.g., Apple Calendar, Google Calendar). |
